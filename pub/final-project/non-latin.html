Introduction

Arabic is one of the most popular yet difficult to master languages in the world! It is one of the six languages used in the United Nations.
It is read right to left and it has 28 different letters. Sometimes, the letters can connect in different ways, making it hard to detect which letters are which.
But don't worry! You'll learn a lot from reading this presentation about the Arabic language. The character encoding I used was UTF-8, the one we use for English normally.
I used this one since it works with Arabic along with English, there was no need to use a different charset or metadata when this one worked!
