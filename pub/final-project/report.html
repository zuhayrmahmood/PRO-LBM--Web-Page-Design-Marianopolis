Report
a.

The editor we picked was Visual Studio Code from Microsoft. It is a lightweight IDE used for many programming languages but also HTML and CSS.
To start using the editor first you must install it from here. This link will take you to the installation
site where you can select the appropriate operating system you have.

b.

There were many helpful, user efficient features. First off, the one I liked the most was the colour coding of the HTML code. This helps organize the code and
makes it much more pleasant to look at from a developper's point of view. Another useful feature was the auto completion. Whenever typing a tag, the auto complete would
fill the rest of the tag if it wasn't a oneshot tag. Also, if you press CTRL + ALT and then use arrow keys, you can write multiple lines at once, increasing productivity.
Lastly, while this isn't exclusive to just VSC, you could use CTRL + arrows to travel between words or sentences with Home and End instead of arrows, to select the travelled distance,
hold shift while travelling!

c.

To be honest, there isn't much that needs to be improved in my opinion. A lot of the features are fairly beginner friendly. One thing I found confusing were the options
for programming, the run and debug features weren't all too useful for me since we aren't programming, so I would stay away from those.

Thank you for reading our report!
