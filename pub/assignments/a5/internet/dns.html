Domain Names

Domain names are addresses people type in their browser to find your website. Various endings are possible for domain names, such as .com, .ca (in Canada), .org, .edu etc. These endings help identify what type of site we are visiting. With .com being the standard, and .org being for organizations and edu being used for educational sites or sites that are tied to education. Domain names are typically stored in DNS, domain name servers. These servers relate each domain name with a specific IP address and direct the user there as requested.
