Software

Software is the "counter" to hardware. While hardware is something physical you can touch, software is not. Software, in other words can be understood as applications. All the apps we have on our phone are software. Many popular software include Microsoft Word, Spotify, Google Chrome etc. Software is a part of our daily lives.
